# Koradio 视觉参考图说明

本目录保存视觉观感参考，不是产品源码或像素回归基线。

## 资产分层

- `01-*.png` 至 `15-*.png`：15 张高保真观感参考，实际尺寸均为 `971 × 1619px`。视觉原型可将其归一化叠加到 `960 × 1600px` 画布，用于比较结构、密度、比例和未被规范覆盖的观感。
- `source/`：用户提供的早期结构图与施工图，用于需求和设计追溯。
- `../assets/prototype/`：当前 HTML / CSS / JavaScript 视觉主源。
- `../assets/baselines/`：从视觉主源派生的 60 张正式截图基线，Prototype 基线为 `960 × 1600px`，用于视觉回归。

## 使用边界

- 产品行为以 `../../docs/prd.md` 为准，用户流程以 `../../docs/user-flow.md` 为准，明确 UI 规则以 `../design.md` 为准。
- 参考图中的端口、路径、时间、歌曲、头像和服务状态均为视觉 fixture，不是产品默认值、真实用户数据或已实现能力。
- 不得从参考图推导权威文档未定义的业务规则，也不得手工修改正式基线以迎合参考图。
- 当前仓库只记录这些文件由项目用户提供或在开发前视觉设计阶段形成；未声明其对外再分发、商业使用或第三方素材授权。需要对外发布时必须另行完成权属核验。
