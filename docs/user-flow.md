## 6. 关键异常分支清单

```text
[本地服务未连接]
  -> 显示独立“本地服务未连接”异常页
  -> 展示 ○ OFFLINE、重试、前往 Settings、诊断信息
  -> 用户重试或进入 Settings 测试服务
  -> 连接成功后返回 Radio

[阻断错误]
  -> 在相关区块内联显示
  -> 提供恢复入口：重试 / 前往 Settings / 修改输入 / 降级继续
  -> 不只依赖 Toast

[未配置 API Key]
  -> 场景发送被阻止
  -> 用户看到缺失项
  -> 跳转 Settings
  -> 保存并测试后重试

[Codex 规划失败]
  -> 保留用户输入
  -> 显示“点击重试”
  -> 用户重试或修改场景

[网易云无结果]
  -> 系统换关键词重试 2 次
  -> 仍无结果时提示换描述
  -> 不创建空节目

[TTS 失败]
  -> 保留文字 DJ
  -> 队列继续播放歌曲
  -> DJ 状态显示“文字串讲”

[TTS 无分句时间戳]
  -> Detail Sheet 保留串讲文本
  -> 按文本长度估算当前句高亮
  -> 播放不受影响

[歌词缺失]
  -> Detail Sheet 显示“暂无歌词”
  -> 波形和播放不受影响

[主题切换失败]
  -> Settings 回滚到上一次主题
  -> Radio 和 Detail Sheet 保持原主题
  -> 提示“主题保存失败”

[DJ 语言切换]
  -> Settings 保存 zh-CN 或 en-GB
  -> 下一次 Codex 规划使用新语言
  -> 当前已生成节目不强制重写串讲

[播放失败]
  -> 当前曲标记不可播放
  -> 自动尝试下一首
  -> 全部失败时提示重新生成

[反馈保存失败]
  -> 播放不中断
  -> UI 状态回滚或提示未保存
  -> 用户可稍后重试

[Reduce Motion]
  -> 关闭持续波形和大部分转场动画
  -> 保留 PLAYING / SPEAKING / THINKING 等文字状态
  -> 播放和流程不受影响
```
